# Milkbelly - A Casa do Milkshake

## Overview
A static website for Milkbelly, a milkshake shop. This is a Brazilian Portuguese website featuring the shop's menu, about section, gallery, and contact information.

## Project Structure
- `index.html` - Main HTML page
- `css/style.css` - Stylesheet
- `js/main.js` - JavaScript for interactivity
- `feed/` - Feed content
- `sabores/` - Flavor images/content
- Various image assets (logos, banners, etc.)

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Fredoka, Nunito, Pacifico)

## Running Locally
The website is served using the `serve` package on port 5000:
```
npx serve -l 5000
```

## Deployment
This is a static website. Deploy by serving all files from the root directory.
